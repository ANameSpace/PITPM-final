# App

Консольное приложение-викторина с хранением вопросов в CSV-файле.

## Возможности
- CRUD вопросов (добавление, просмотр, редактирование, удаление)
- Запуск викторины с подсчётом правильных ответов
- Фильтрация по категории и уровню сложности (1–5)
- Сравнение ответов без учёта регистра

## Запуск
```bash
pip install -r requirements.txt
python main.py
```

## Формат данных (CSV)
| Поле       | Тип | Описание                                 |
|------------|-----|------------------------------------------|
| id         | int | Уникальный идентификатор                 |
| question   | str | Текст вопроса                            |
| answer     | str | Правильный ответ                         |
| difficulty | int | Сложность 1–5                            |
| category   | str | "Категория (География Математика и др.)" |

## Схема взаимодействия
Пользователь → меню (main.py) → бизнес-логика (quiz.py) → хранилище (storage.py) → файл questions.csv

## Архитектура
```
main.py          ← CLI-меню (точка входа)
src/
├── models.py    ← модель Question (dataclass)
├── storage.py   ← операции с CSV (read/write/filter)
└── quiz.py      ← бизнес-логика викторины
tests/           ← модульные тесты (pytest)
data/
└── questions.csv ← хранилище (создаётся автоматически)
```

## Импорт вопросов из CSV

Есть два способа:

**1. Прямая замена:**
Замените `data/questions.csv` вашим файлом (сохраните заголовок: `id,question,answer,difficulty,category`).

**2. Объединение:**
Откройте ваш CSV, скопируйте строки без заголовка и вставьте в конец `data/questions.csv`. Пересчитайте ID вручную.
